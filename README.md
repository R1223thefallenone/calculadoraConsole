# App de Compras en JavaScript

Aplicacion simple en JavaScript que se ejecuta en la consola del navegador.
Permite ingresar productos, calcular precios, IVA y total usando funciones,
arreglos, objetos y estructuras de control.

## Tecnologias
- JavaScript

## Como usar
1. Abrir el navegador
2. Abrir la consola (F12)
3. Pegar el codigo
4. Seguir las instrucciones por pantalla

## Conceptos aplicados
- Variables
- Funciones
- Arreglos
- Objetos
- Condicionales
- Ciclos
